================================================================================
                    NEPTUNE AI — VOICE MODE IMPLEMENTATION PLAN
================================================================================

GOAL: Real-time voice conversation with the Neptune AI agent, just like 
ChatGPT's voice mode. User speaks → agent processes → agent replies with voice.

================================================================================
APPROACH OPTIONS (Pick One)
================================================================================

OPTION A: Browser STT + Server TTS (RECOMMENDED — simplest, cheapest)
  - Speech-to-Text: Browser's Web Speech API (free, runs in browser)
  - Text-to-Speech: OpenAI TTS API ($15/1M chars) or ElevenLabs
  - Flow: User speaks → browser transcribes → send text to backend → 
          get text response → server generates audio → stream to frontend
  - Pros: Cheap, easy to implement, works with your existing chat flow
  - Cons: Web Speech API needs Chrome/Edge (no Firefox), slight delay

OPTION B: Full Server-Side (Whisper + TTS)  
  - Speech-to-Text: OpenAI Whisper API ($0.006/min) — record audio, send to API
  - Text-to-Speech: OpenAI TTS API or ElevenLabs
  - Flow: User speaks → record audio blob → send to backend → 
          Whisper transcribes → LLM processes → TTS generates audio → stream back
  - Pros: Works in ALL browsers, more accurate transcription
  - Cons: More latency (upload audio → transcribe → process → generate audio)

OPTION C: OpenAI Realtime API (Most ChatGPT-like, but expensive)
  - Uses WebSocket connection with GPT-4o-realtime
  - Native voice-to-voice with interruptions, emotions, natural flow
  - Pros: Most natural, lowest latency, supports interruptions
  - Cons: Very expensive ($5/1M input tokens audio), needs OpenAI API key 
          (not NEAR AI), completely separate from your current LLM flow

>>> RECOMMENDATION: Option A for v1, upgrade to B later if needed.

================================================================================
COST ESTIMATES
================================================================================

OpenAI TTS (tts-1):
  - $15 per 1 million characters
  - Average response: ~200 chars = $0.003 per response
  - 1000 voice interactions/day = ~$3/day

OpenAI Whisper (if using Option B):
  - $0.006 per minute of audio
  - Average voice input: 10 seconds = $0.001 per input
  - Very cheap

ElevenLabs (alternative, more natural voices):
  - Free tier: 10,000 chars/month
  - Starter: $5/month for 30,000 chars
  - More expensive but MUCH more natural-sounding

Browser Web Speech API (Option A):
  - FREE — runs entirely in the browser
  - No API costs for speech recognition

================================================================================
BROWSER COMPATIBILITY
================================================================================

Web Speech API (for STT):
  ✅ Chrome (desktop + Android)
  ✅ Edge  
  ✅ Safari (iOS 14.5+, macOS)
  ❌ Firefox (not supported)
  
  Fallback for Firefox: Show text input with message 
  "Voice mode requires Chrome, Edge, or Safari"

Audio playback (for TTS):
  ✅ All modern browsers support <audio> with MP3

================================================================================
ALTERNATIVE: DEEPER VOICE INTEGRATION
================================================================================

If you want even MORE ChatGPT-like voice mode in the future:

1. WebRTC + Server-Sent Events
   - Real-time bidirectional audio streaming
   - Lower latency, more natural conversation flow
   - Significantly more complex to implement

2. OpenAI Realtime API
   - Native voice-to-voice, no STT/TTS pipeline needed
   - Supports interruptions mid-sentence
   - Would need separate WebSocket connection
   - ~$100/day for 1000 conversations (expensive)

3. Ultravox / LiveKit
   - Open-source alternatives for realtime voice AI
   - Can run your own models
   - More control, lower cost at scale

>>> Start with Option A (Browser STT + OpenAI TTS), it's 90% of the 
    ChatGPT voice experience at 1% of the cost.
